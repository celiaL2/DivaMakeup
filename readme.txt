============================================
 DiveMakeup — Boutique de maquillage en ligne 
============================================

DESCRIPTION DU SITE
-------------------
DivaMakeup est un mini site e-commerce , spécialisée dans la vente de produits de maquillage et de beauté en offrirant les derniers tendance.
Le site propose un catalogue dynamique filtrable par catégorie,
un système d'authentification simulé, un panier interactif et
un formulaire de commande avec validation côté client.



INSTRUCTIONS D'UTILISATION
---------------------------
1. Ouvrir le fichier index.html dans un navigateur moderne.
2. Explorer le catalogue via la page Catalogue (filtrage par catégorie).
3. Ajouter des articles au panier en cliquant sur "Add".
4. Accéder au panier via l'icône 🛒 ou le menu "Mon Panier".
5. Se connecter via la page Connexion ou créer un compte via Inscription.

6.Passer une commande via Commande.
